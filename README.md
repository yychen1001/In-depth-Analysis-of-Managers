# In-depth Analysis of Managers

单管理人深度分析报告静态站点。

## 1.0 标准交付

- `庄贤投资深度分析报告.html`：网页阅读版本，支持目录导航、目录折叠、目录拖拽、返回顶部、打印、下载 Word、下载 MD。
- `庄贤投资深度分析报告.docx`：正式流转 Word 版本。
- `庄贤投资深度分析报告.md`：Markdown 内容母版。
- `单管理人深度分析报告生成PRD.md`：可复用到其他管理人的报告生成 PRD。

## GitHub Pages

启用方式：

1. 进入仓库 `Settings`。
2. 打开 `Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后访问：
   `https://yychen1001.github.io/In-depth-Analysis-of-Managers/`

站点入口为 `index.html`，会自动跳转到报告网页。
